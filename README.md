Проект учебной практики (RIPEMD-160)

Данный репозиторий содержит программу для хеширования данных с использованием алгоритма RIPEMD-160, а также документацию, разработанную в рамках учебной практики по направлению «Информационная безопасность».

Структура проекта:

project/
├── src/           # Исходные файлы программы
│   ├── ripemd160.py       # Реализация алгоритма хэширования RIPEMD-160 (используя библиотеку `hashlib`)
│   ├── encrypt.py         # Модуль шифрования (точнее хэширование данных)
│   ├── decrypt.py         # Фиктивный модуль расшифровки (так как алгоритм необратимый)
│   └── gui.py             # Графический интерфейс приложения
├── data/          # Тестовые наборы данных
│   ├── test_set_1.txt     # Первый тестовый набор
│   ├── test_set_2.txt     # Второй тестовый набор
│   └── control_set.txt    # Контрольный набор данных
├── docs/          # Документация проекта
│   ├── AS_IS.txt          # Текущее состояние реализации
│   ├── TO_BE.txt          # Планируемое будущее развитие
│   └── algorithm_scheme.png # Диаграмма схемы алгоритма
├── PI_Practice_Report.docx # Отчёт по практике (PI-Practice Report)
└── README.md      # Описание проекта и инструкции по запуску

Запуск программы

Python-версия
Требуется Python 3.9+  
Установите зависимости:

pip install -r requirements.txt


Запуск GUI:

python src/gui.py


Функциональность программы
- Ввод текста вручную.
- Генерация хеша RIPEMD-160.
- Проверка целостности данных.
- Интуитивно понятный GUI.

Наборы данных
В папке `data/` расположены тестовые файлы, использованные в ходе практики.

Документация
В папке `docs/` находятся:
- AS-IS и TO-BE схемы,
- схема алгоритма RIPEMD-160,
- техническое задание по ГОСТ 34.602–2020,
- другие материалы.
