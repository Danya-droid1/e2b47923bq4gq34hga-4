Проект учебной практики (RIPEMD-160)

Данный репозиторий содержит программу для хеширования данных с использованием алгоритма RIPEMD-160, а также документацию, разработанную в рамках учебной практики по направлению «Информационная безопасность».

Структура проекта:

project/
├─ src/
│ ├─ ripemd160.py # Реализация алгоритма (через hashlib)
│ ├─ encrypt.py # Модуль "шифрования" (хеширования)
│ ├─ decrypt.py # Заглушка (алгоритм необратим)
│ └─ gui.py # Графический интерфейс
├─ data/
│ ├─ test_set_1.txt
│ ├─ test_set_2.txt
│ └─ control_set.txt
├─ docs/
│ ├─ AS_IS.txt
│ ├─ TO_BE.txt
│ ├─ algorithm_scheme.png
├─ PI_Practice_Report.docx
├─ README.md



Запуск программы

Python-версия
Требуется Python 3.9+  
Установите зависимости:

pip install -r requirements.txt


Запуск GUI:

python src/gui.py


Функциональность программы
- Ввод текста вручную.
- Генерация хеша RIPEMD-160.
- Проверка целостности данных.
- Интуитивно понятный GUI.

Наборы данных
В папке `data/` расположены тестовые файлы, использованные в ходе практики.

Документация
В папке `docs/` находятся:
- AS-IS и TO-BE схемы,
- схема алгоритма RIPEMD-160,
- техническое задание по ГОСТ 34.602–2020,
- другие материалы.
